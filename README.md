Author: Kathryn F. McDaniel - kathryn_mcdaniel@brown.edu

These files are the R code files for data analysis for the McDaniel et. al., manuscript currently in submission.

Files are named by the figure and figure panel for which the analysis was performed.

RNA-Sequencing data is organized by experiment. Data from H1 is for Figure 4 and Supplemental Figure 2A-C;E. Data from H2 is for Supplemental Figure 2D.
Raw fastq files and both raw and normalized counts for H1 can be found on GEO at GSE329652. 
Raw fastq files and both raw and normalized counts for H2 can be found on GEO at GSE329700.

Both experiments are contained on GEO under SuperSeries Accession GSE329701.
